# Currency Converter

Live: https://unkyo.github.io/currency-converter/

## Features
- Converts: RUB, USD, EUR, KZT, GEL, GBP
- Dark/light/system theme toggle
- Favorites (pairs) stored locally
- Offline fallback (uses cached last rate per base)
